PREDICTIVE MODELING USING MACHINE LEARNING

Predict Customer Lifetime Value LTV-  B2C SAAS PRODUCT SUBSCRIPTION MODEL

Business Use Case - For an organisation using Business to Consumer model for their products like Netflix, Disney+ , Spotify,Amazon Prime would want to know how much revenue a newly signed customer is generating over their first year.
This key metric helps marketing,sales and product teams to answer business questions like -  

1. How much money should we be spending on acquiring the customer ?
2. What Product features can be changed to engage more customers - Applying Product Stickiness.
3. What customers are driving the maximum LTV and how can we provide more value to these customers - Can be used to drive Upsell strategies.

PROJECT OVERVIEW -
As part of this project, I simulated a real world B2C customer dataset for 5k customers across 4 regions in the USA with their engagement data like active days per week and monthly session count to build Linear Regression Model using Python and Machine Learning concepts.

In this project, python is used to preprocess, summarize, and visualize customer engagement data. 
The goal is to analyzethe relationship between customer demographics , engagaement and their lifetime revenue for 12 months to help a B2C organisation optimize its marketing and sales strategies. 
The tasks include data extraction from a JSON file using pandas, pre-processing, visualization, statistical modeling, and recommendations based on the results.
